# Sintetica Factory Showcase

Uma demonstração do sistema **Sintetica Factory** - transformando dados em experiências narrativas imersivas.

## O que é isso?

Este é um mini-scrollytelling que demonstra as capacidades do Sintetica Factory:

- **Sistema de Temperatura** - Jornada emocional através de cores e atmosfera
- **Estrutura 5 Atos** - Narrativa dramática com arco completo
- **Breathers** - Pausas cognitivas entre seções densas
- **Hero Socrático** - Revelação em estágios ao estilo Rosling
- **35+ Componentes** - Biblioteca completa para experiências

## Estrutura Narrativa

```
PROLOGUE [COLD] → Hero com revelação Socrática
    ↓
ACT 1 [COOL] → "O Problema"
    ↓ Breather: Pergunta
ACT 2 [NEUTRAL] → "A Solução"
    ↓ Breather: Estatística gigante
ACT 3 [WARM] → "Os Componentes"
    ↓ Breather: Citação
ACT 4 [HOT] → "O Resultado"
    ↓ Breather: Silêncio
ACT 5 [COOL] → "Os Agentes"
    ↓
EPILOGUE [COLD] → Call to action
```

## Stack

- **React 18** + **TypeScript**
- **Tailwind CSS** - Design system
- **Vite** - Build tool
- **Lucide React** - Icons

## Rodar localmente

```bash
npm install
npm run dev
```

Abra http://localhost:5173

## Sistema de Temperatura

| Temperatura | Mood | Uso |
|-------------|------|-----|
| **COLD** | Impacto, seriedade | Aberturas, encerramentos |
| **COOL** | Informativo | Contexto, explicações |
| **NEUTRAL** | Técnico | Dados densos |
| **WARM** | Humanização | Histórias, casos |
| **HOT** | Urgência | Tensão, alertas |
